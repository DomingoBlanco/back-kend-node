#Notas Prueba Back-End para prueba sophos
```
-GIT HUB donde esta el proyecto   = https://github.com/DomingoBlanco/back-kend-node
-GIT CLONE = https://github.com/DomingoBlanco/back-kend-node.git
-Clonar el codigo fuente con el comando git clone
-Tener node instalado
-Tener NPM instalado
-Tener Mysql instalado
-importar el script de base de datos base_datos_sophos.sql a la base de datos mysql se encuentra en git
-colocarse en la raiz del proyecto cd no-pain-no-gain
-ejecutar el comando npm install
-para ejecutar el proyecto colocarse en la raiz del proyecto cd no-pain-no-gain y ejecutar el comando node app
- Collection de Postman esta en dos formatos en un archivo de nombre no-pain-no-gain.postman_collection.json y el link https://www.postman.com/collections/02c6b7a27d4cfa3c0883
```
